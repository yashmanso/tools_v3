## How to Update This Document

When adding or updating prerequisites:

1. Update the tool entry in this markdown file
2. Update the corresponding entry in `website/app/lib/prerequisites-data.ts`
3. Ensure the tool slug matches exactly (use kebab-case)
4. Include all three types of prerequisites: concepts, tools, and skills
5. Specify skill level requirements accurately
6. Add links to related tools when applicable
